# 九田家·春节运营数据智能看板

基于 114 家门店、65 个城市的春节运营数据，分析美团 vs 抖音双平台表现，提供数据驱动的运营决策建议。

🔗 **在线看板：** https://yuchuanwang001-source.github.io/jiutianjia-dashboard/

## 核心发现

- 🚨 美团下滑 24%（438万→332万），抖音增长 149%（78万→196万）
- 📊 抖音增长几乎全部来自美团流失存量，而非新增客源
- 🏪 门店级差高达 117 倍（最高店 19.8万 vs 最低店 1692元）
- 🎯 仍有 17% 门店抖音完全空白，增长空间明确

## 技术栈

- Chart.js（数据图表）
- 纯 CSS 动画（粒子效果、滚动触发、数字计数器）
- 响应式布局（适配电脑/平板/手机）
